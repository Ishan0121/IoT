Experiment 05

Ultrasonic Sensor
Handling with Arduino

Measure distance with HC-SR04 using pulseIn(). Implement danger/caution/safe zones with LEDs and non-blocking buzzer alerts.

🎯 HC-SR04 · pulseIn · Sound Speed Intermediate TRIG=9 · ECHO=10

⚡ How HC-SR04 Works

1. Trigger Pulse
Send 10μs HIGH on TRIG pin → sensor emits 8× 40kHz ultrasonic bursts
→
2. Sound Travels
Sound wave travels to object and reflects back toward sensor
→
3. Echo Pulse
ECHO pin goes HIGH for duration = round-trip travel time
→
4. Calculate
distance = (duration × 0.0343) ÷ 2
(speed of sound ≈ 343 m/s)

🟢 Simple Solution — Basic Distance Measurement

⭐ Start Here — single reading per loop, print to Serial Monitor. Just TRIG + ECHO pins.
150 cm
Serial Monitor
Simulates basic pulseIn() and delay(500) reading loop
simple.ino — Exp 05 (Beginner)
// Simple HC-SR04 Distance Measurement
// TRIG=9, ECHO=10

const int TRIG=9, ECHO=10;

void setup() {
  Serial.begin(9600);
  pinMode(TRIG, OUTPUT);
  pinMode(ECHO, INPUT);
}

void loop() {
  // Send 10µs trigger pulse
  digitalWrite(TRIG, LOW);  delayMicroseconds(2);
  digitalWrite(TRIG, HIGH); delayMicroseconds(10);
  digitalWrite(TRIG, LOW);

  // Measure echo pulse width
  long duration = pulseIn(ECHO, HIGH);

  // Convert to cm: speed of sound = 0.0343 cm/µs, divide by 2 (round trip)
  float distance = (duration * 0.0343) / 2;

  Serial.print("Distance: ");
  Serial.print(distance);
  Serial.println(" cm");

  delay(500); // Wait before next reading
}

💾 Advanced Solution — Averaging + Zones + Non-blocking Buzzer

🚀 Advanced — 5-sample averaging, danger/caution/safe LED zones, non-blocking buzzer alerts.
program.ino — Exp 05
// Exp 05 — HC-SR04 Ultrasonic Sensor
// TRIG=9, ECHO=10, Green=4, Yellow=5, Red=6, Buzzer=7

const int TRIG=9,ECHO=10,GREEN_LED=4,YELLOW_LED=5,RED_LED=6,BUZZ=7;
const int DANGER_CM=20, CAUTION_CM=40;

unsigned long lastBuzzToggle=0;
bool buzzState=false;
int buzzInterval=0;

// Get raw distance in cm (returns -1 on timeout)
long getRaw(){
  digitalWrite(TRIG,LOW); delayMicroseconds(2);
  digitalWrite(TRIG,HIGH); delayMicroseconds(10);
  digitalWrite(TRIG,LOW);
  long dur=pulseIn(ECHO,HIGH,25000);
  return dur==0 ? -1 : (dur*0.0343)/2;
}

// Average of 5 readings for stability
long getDistance(){
  long sum=0; int valid=0;
  for(int i=0;i<5;i++){
    long d=getRaw();
    if(d>0 && d<=400){sum+=d;valid++;}
    delay(15);
  }
  return valid>0 ? sum/valid : -1;
}

void allLedsOff(){
  digitalWrite(GREEN_LED,LOW);digitalWrite(YELLOW_LED,LOW);digitalWrite(RED_LED,LOW);
}

void updateAlerts(long dist){
  allLedsOff();
  if(dist<DANGER_CM){
    digitalWrite(RED_LED,HIGH); buzzInterval=100;
    Serial.print(F("[DANGER] "));
  } else if(dist<CAUTION_CM){
    digitalWrite(YELLOW_LED,HIGH); buzzInterval=400;
    Serial.print(F("[CAUTION] "));
  } else {
    digitalWrite(GREEN_LED,HIGH); buzzInterval=0;
    digitalWrite(BUZZ,LOW);
    Serial.print(F("[SAFE] "));
  }
  Serial.print(dist); Serial.println(F(" cm"));
}

// Non-blocking buzzer beeper
void updateBuzzer(){
  if(buzzInterval==0)return;
  unsigned long now=millis();
  if(now-lastBuzzToggle>=(unsigned long)buzzInterval){
    lastBuzzToggle=now; buzzState=!buzzState;
    digitalWrite(BUZZ,buzzState?HIGH:LOW);
  }
}

void setup(){
  Serial.begin(9600);
  pinMode(TRIG,OUTPUT); pinMode(ECHO,INPUT);
  pinMode(GREEN_LED,OUTPUT); pinMode(YELLOW_LED,OUTPUT);
  pinMode(RED_LED,OUTPUT); pinMode(BUZZ,OUTPUT);
  allLedsOff(); digitalWrite(BUZZ,LOW);
  Serial.println(F("=== HC-SR04 Distance Sensor ==="));
}

void loop(){
  long dist=getDistance();
  if(dist<0){Serial.println(F("[--] Out of range"));allLedsOff();buzzInterval=0;}
  else updateAlerts(dist);
  updateBuzzer(); delay(50);
}

📝 Notes & Extras

💡 Speed of Sound Formula
distance = (duration_µs × 0.0343) / 2
Sound travels at ≈343 m/s = 0.0343 cm/µs. Divide by 2 because pulseIn() measures the round-trip (to object AND back). HC-SR04 range: 2–400 cm, beam angle: ~15°.
⚠️ Sensor Noise & Averaging
Single readings can spike due to reflections or interference. Averaging 5 consecutive readings smooths noise significantly. Add a timeout (pulseIn(pin, HIGH, 25000)) so the sketch doesn't hang if no echo returns (out of range).
🔧 Non-Blocking Buzzer
Instead of tone()/delay() which block the CPU, track lastBuzzToggle and toggle the buzzer pin when enough time has passed. This lets the sensor still take readings while the buzzer beeps — crucial for real-time responsiveness.
📐 HC-SR04 Pinout
VCC → 5V  |  GND → GND
TRIG → Digital output (send 10μs pulse)
ECHO → Digital input (read pulse width)
Do NOT connect ECHO directly to 5V logic if using 3.3V boards — use a voltage divider.

🔧 Key Functions

pulseIn(pin, value, timeout)
Measures the duration (µs) of a pulse on a pin. Waits for the pin to go to value (HIGH), then times how long it stays there. timeout (µs) prevents infinite hang.
delayMicroseconds(us)
Pause for the specified number of microseconds. Used to generate the precise 10μs TRIG pulse required by HC-SR04.
getRaw() / getDistance()
Custom functions: getRaw() gets one sample, getDistance() averages 5 valid samples for noise rejection. Returns -1 if no valid reading.
updateBuzzer()
Non-blocking buzzer manager — checks millis() to toggle pin at the right interval without using delay().
