Servo Motor & Stepper
Motor Handling

Control SG90 servo (sweep + pot control) and 28BYJ-48 stepper with ULN2003 driver. Understand PWM pulses vs step sequences.

🎯 Servo.h · Stepper.h · SG90 · 28BYJ-48 Intermediate Pin 9 (Servo)

🧠 Theory, Logic & Tactics

📖 Theory
  • Servo motors use standard PWM (1ms-2ms pulses) for precise angular positioning (0-180 degrees).
  • Stepper motors rotate in discrete steps via electromagnetic coils, offering continuous rotation with high holding torque.
  • The ULN2003 driver chip amplifies the low-current Arduino signals to drive the stepper coils.
⚙️ Logic
  • The <Servo.h> library abstracts complex PWM timing into a simple write(angle) command.
  • The <Stepper.h> library converts a desired number of steps into a precise 4-phase coil activation sequence.
  • Mapping a 10-bit analog potentiometer (0-1023) to an 8-bit servo angle (0-180) requires the map() function.
♟️ Tactics
  • Motors draw inductive current spikes that can reset the Arduino. Always power them from an external 5V/12V supply, sharing a common GND.
  • Do not manually force a servo horn to turn while it is powered, as you will strip the internal plastic gears.
  • If a stepper motor vibrates but fails to turn, double-check your coil wiring sequence (often IN1-IN3-IN2-IN4).

simple.ino — Exp 06 Servo (Beginner)
// Simple Servo Sweep
// Circuit: Servo signal wire → Pin 9

#include <Servo.h>

Servo myServo;

void setup() {
  myServo.attach(9);  // Connect servo to Pin 9
}

void loop() {
  // Sweep from 0 to 180 degrees
  for(int angle=0; angle<=180; angle++) {
    myServo.write(angle);
    delay(15);
  }
  // Sweep back from 180 to 0
  for(int angle=180; angle>=0; angle--) {
    myServo.write(angle);
    delay(15);
  }
}

💾 Advanced Solution — Pot Control + Smooth Move

🚀 Advanced — potentiometer-driven angle, smooth moveTo() function, dead-band jitter prevention.
program.ino — Servo (Part A)
#include <Servo.h>

const int SERVO_PIN=9, POT_PIN=A0;
Servo myServo;
int currentAngle=90;

// Smooth move to target angle
void moveTo(int target, int stepDelay=15){
  int step=(target>currentAngle)?1:-1;
  while(currentAngle!=target){
    currentAngle+=step;
    myServo.write(currentAngle);
    Serial.println(currentAngle);
    delay(stepDelay);
  }
}

void setup(){
  Serial.begin(9600);
  myServo.attach(SERVO_PIN);
  myServo.write(90); delay(500);
  Serial.println(F("=== Servo Motor Control ==="));
  // Auto sweep for first 5 seconds
  moveTo(180,10); moveTo(0,10);
  Serial.println(F("Sweep done. Now using Potentiometer."));
}

void loop(){
  // Pot controls angle in real-time
  int potVal=analogRead(POT_PIN);
  int target=map(potVal,0,1023,0,180);
  if(abs(target-currentAngle)>2){
    myServo.write(target); currentAngle=target;
    Serial.print(F("Angle: ")); Serial.println(target);
  }
  delay(50);
}

💾 Program — Part B: Stepper

program.ino — Stepper (Part B)
#include <Stepper.h>

const int STEPS_PER_REV=2048;
// IN1=8,IN3=10,IN2=9,IN4=11 (pin ORDER matters!)
Stepper myStepper(STEPS_PER_REV,8,10,9,11);

void rotateDeg(float degrees, bool cw){
  long steps=(degrees/360.0)*STEPS_PER_REV;
  if(!cw) steps=-steps;
  Serial.print(cw?F("CW  "):F("CCW "));
  Serial.print(abs(degrees)); Serial.println(F("°"));
  myStepper.step(steps);
}

void setup(){
  Serial.begin(9600);
  myStepper.setSpeed(10); // 10 RPM
  Serial.println(F("=== 28BYJ-48 Stepper ==="));
}

void loop(){
  rotateDeg(360,true); delay(1000);
  rotateDeg(90,false); delay(500);
  rotateDeg(180,true); delay(500);
  rotateDeg(90,false); delay(1000);
}

📝 Notes & Extras

💡 How Servo PWM Works
SG90 servo reads a 50Hz PWM signal (20ms period). Pulse width sets angle: 544µs=0°, 1500µs=90°, 2400µs=180°. The Servo.h library handles all timing — just call myServo.write(0–180).
🔄 Stepper Step Sequence
28BYJ-48 is a 4-phase unipolar stepper. The ULN2003 driver energizes coils in sequence: 1000 → 1100 → 0100 → 0110 → 0001 → ... (half-step). 2048 steps = 1 full revolution. The pin ORDER in Stepper() constructor is critical: IN1, IN3, IN2, IN4.
⚠️ Servo Power Warning
Do NOT power a servo directly from the Arduino 5V pin if it's under load — it can draw 500mA+ causing brownouts and resets. Power the servo from an external 5V supply (share GND with Arduino). The signal wire alone goes to Pin 9.
📐 map() Function
map(value, inMin, inMax, outMin, outMax) linearly maps a value from one range to another. Potentiometer gives 0–1023; servo needs 0–180. The dead-band check (abs(target-current)>2) prevents servo jitter from ADC noise.

🔧 Key Functions

myServo.attach(pin)
Attach the Servo object to a PWM pin. Starts generating the 50Hz PWM signal on that pin.
myServo.write(angle)
Move servo to angle 0–180°. Library translates to appropriate pulse width (544–2400µs).
myStepper.setSpeed(rpm)
Set stepper speed in RPM. For 28BYJ-48: 1–15 RPM is practical. Higher speeds lose torque.
myStepper.step(steps)
Rotate a number of steps. Positive = CW, negative = CCW. Blocking call — motor must finish before loop continues.
map(val, iMin, iMax, oMin, oMax)
Linearly re-maps a value from input range to output range. Essential for converting potentiometer 0–1023 to servo 0–180°.
